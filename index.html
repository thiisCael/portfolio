Task Manager

Apresento um Task Manager em Java, totalmente operável via terminal. O sistema permite adicionar, editar, excluir e marcar tarefas como concluídas. Além disso, fornece estatísticas que acompanham o progresso das atividades, oferecendo visão clara do seu desempenho. Todos os dados são armazenados localmente, garantindo que suas tarefas fiquem salvas de forma segura e organizada.
